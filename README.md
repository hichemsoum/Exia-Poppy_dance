# Exia-Poppy_dance
A challenge to make a robot dance a little

---


## Requirements :
To run you will need :
- python 3
- [coppeliasim](https://www.coppeliarobotics.com/downloads) 
- Poppy ergo jr library

```bash
pip install poppy-ergo-jr
```

##Usage :
    1-open coppeliasim
    2- run the command
```bash
python3 move.py
```
